# TimeScreen Control — Родительский контроль для Windows

Программа для ограничения времени использования компьютера. Архитектура: служба Windows (SYSTEM) + GUI-приложение. Не требует прав администратора для повседневной работы (только для установки).

## Возможности

- ✅ **Графический интерфейс** — вкладки «Общие», «Пользователи», «Интервалы», «Таймер»
- ✅ **Служба Windows** — мониторинг времени под SYSTEM, автозапуск, нельзя убить через Task Manager
- ✅ **Гибкое расписание** — интервалы с настройкой дней недели, включая overnight (22:00–08:00)
- ✅ **Блокировка всех мониторов** — `EnumDisplayMonitors`, каждый экран получает полноэкранное окно
- ✅ **Перетаскиваемый таймер** — обратный отсчёт до блокировки/разблокировки, позиция сохраняется
- ✅ **Авторазблокировка** — экран сам закроется, когда настанет разрешённое время
- ✅ **Grace-период** — 10 минут без повторной блокировки после ввода пароля
- ✅ **Разблокировка по паролю** — bcrypt-хэширование (12 раундов) с солью
- ✅ **Admin Unlock** — вход как администратор Windows сбрасывает блокировку
- ✅ **Multi-user** — выбор контролируемых пользователей Windows через PowerShell
- ✅ **Защита от взлома** — SHA-256 хэш целостности конфига, при повреждении — permanent lockdown
- ✅ **Кнопки выключения** — завершение работы, перезагрузка, сон прямо с экрана блокировки
- ✅ **Без зависимостей** — собранный EXE не требует Python, прав администратора, интернета

### Два исполняемых файла

| Файл | Формат | Назначение |
|------|--------|------------|
| `TimeScreenControl.exe` | `--onefile --windowed` | GUI: настройки, таймер, экран блокировки |
| `TimeScreenService.exe` | `--onedir --console` | Служба Windows под SYSTEM |

**Почему `--onedir` для службы:** PyInstaller `--onefile` bootloader перезапускает сам себя (меняет PID) для ASLR на Windows 10/11. SCM отвергает подключение с другого PID → Error 1053. `--onedir` не перезапускается, PID сохраняется.

## Быстрый старт

### Сборка из исходников (компьютер разработчика)

```bash
cd TimeScreenControl
build.bat
```

Скрипт автоматически:
1. Проверит/установит PyInstaller
2. Соберёт `TimeScreenControl.exe` (onefile, windowed)
3. Соберёт `TimeScreenService\` (onedir)
4. Упакует всё в `dist\Release\`

### Установка (целевой компьютер)

1. Скопируйте папку `dist\Release\` на целевую машину
2. Запустите `install.bat` **от имени администратора**
3. Служба установится и запустится автоматически
4. Ярлык появится в меню «Пуск»

### Первый запуск

1. Откройте «TimeScreen Control» из меню «Пуск»
2. Программа предложит задать пароль администратора (минимум 4 символа)
3. Вкладка **Общие** — вкл/выкл защиты, смена пароля
4. Вкладка **Пользователи** — выбор контролируемых пользователей
5. Вкладка **Интервалы** — настройка разрешённого времени по дням недели
6. Вкладка **Таймер** — отображение обратного отсчёта

### Полное удаление

```batch
install\uninstall.bat   ← запустить от администратора
```

## Как это работает

```
┌──────────────────────────────────────────────────┐
│  СЛУЖБА (TimeScreenService.exe, SYSTEM)           │
│  Session 0, нельзя убить через Task Manager        │
│                                                    │
│  Каждые 60 сек:                                   │
│  1. Читает pc_config.json                         │
│  2. Проверяет, включена ли защита                  │
│  3. Проверяет, контролируется ли текущий           │
│     пользователь                                  │
│  4. Проверяет время                               │
│  5. Если запрещено → запускает LockScreen         │
│     в сессии пользователя через                   │
│     CreateProcessAsUser                           │
│  6. Если разрешено → убивает LockScreen            │
└──────────────┬───────────────────────────────────┘
               │
               ▼
┌──────────────────────────────────────────────────┐
│  LockScreen (TimeScreenControl.exe --locker-mode) │
│  Сессия пользователя, GUI                         │
│                                                    │
│  - Полноэкранное окно на всех мониторах            │
│  - Кнопки: разблокировка, выкл, перезагрузка, сон  │
│  - Каждые 3 сек проверяет авторазблокировку        │
│  - Если время стало разрешённым → самозакрытие     │
│  - Перехватывает Alt+F4, Escape, Win+D            │
└──────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────┐
│  TimerOverlay (TimeScreenControl.exe --timer-mode)│
│  Сессия пользователя, always-on-top               │
│                                                    │
│  - Зелёный: обратный отсчёт до блокировки          │
│  - Жёлтый: grace-период                           │
│  - Красный: компьютер заблокирован                 │
│  - Перетаскивается мышью                          │
│  - Правый клик: скрыть/сбросить позицию            │
└──────────────────────────────────────────────────┘
```

## Безопасность

| Действие | Требуется |
|----------|-----------|
| Изменение расписания | Пароль |
| Вкл/выкл защиты | Пароль |
| Смена пароля | Старый пароль |
| Разблокировка экрана | Пароль |
| Admin Unlock (Windows) | Права админа |
| Установка/удаление службы | UAC |
| Удаление программы | Пароль |
| Целостность конфига | SHA-256 хэш (повреждение = lockdown) |

**Дополнительные меры защиты:**
- Пароли хэшируются **bcrypt** (12 раундов) — защита от brute-force
- Конфиг в `%PROGRAMDATA%` — запись только от админа
- Установка в `%PROGRAMFILES%` — защита от подмены EXE
- Глобальный мьютекс `Global\TimeScreenLock` — защита от гонки процессов
- Нет recovery-режима и backdoor

## Решение проблем

### Служба не запускается (Error 1053)
- Убедитесь, что `TimeScreenService.exe` собран с `--onedir` (не `--onefile`)
- Проверьте, что `_internal\` лежит рядом с `TimeScreenService.exe`
- Запустите `uninstall.bat` и переустановите службу

### Служба висит в START_PENDING
- Старая служба удалена не полностью → перезагрузите ПК
- Или удалите через реестр: `HKLM\SYSTEM\CurrentControlSet\Services\TimeScreenControl`

### Таймер не показывается
- Защита не запущена → откройте настройки и включите защиту

### Мерцание экрана
- В актуальной версии исправлено: `-alpha` вместо `-transparentcolor`, `CREATE_NO_WINDOW` для subprocess

### Блокируется только один монитор
- Используется `EnumDisplayMonitors` — все мониторы блокируются

### Забыли пароль
- Удалите `%PROGRAMDATA%\TimeScreen\pc_config.json` и задайте пароль заново
- Или используйте Admin Unlock на экране блокировки
