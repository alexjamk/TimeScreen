# TimeScreen Control — Родительский контроль для Windows

Программа для ограничения времени использования компьютера. Полноценный графический интерфейс, не требует прав администратора, работает как standalone `.exe` без установленного Python.

## Возможности

- ✅ **Графический интерфейс** — главное окно настроек, без консоли
- ✅ **Гибкое расписание** — несколько временных интервалов (включая через полночь)
- ✅ **Блокировка всех мониторов** — каждый экран получает отдельное полноэкранное окно
- ✅ **Таймер-индикатор** — маленькое окно поверх всех окон с обратным отсчётом
- ✅ **Grace-период** — 10 минут без повторной блокировки после ввода пароля
- ✅ **Разблокировка по паролю** — SHA-256 хэширование
- ✅ **Кнопки выключения/перезагрузки** на экране блокировки
- ✅ **Встроенный установщик** — установка в папку пользователя, ярлыки, автозагрузка
- ✅ **Защита настроек** — пароль требуется для изменения расписания, вкл/выкл защиты, смены пароля
- ✅ **Без зависимостей** — на целевом ПК не нужен Python, права администратора, интернет

## Быстрый старт (GUI-версия)

### На компьютере разработчика (где есть Python):

```bash
# 1. Собрать GUI .exe
cd ParentalControlStandalone
python -m PyInstaller --windowed --onefile --name "TimeScreenControl" ^
    --hidden-import=tkinter --hidden-import=tkinter.messagebox ^
    --hidden-import=tkinter.ttk --hidden-import=tkinter.simpledialog ^
    parental_control_gui.py

# 2. Скопировать в установочный пакет
copy dist\TimeScreenControl.exe TimeScreen_Installer\
```

Готовый пакет лежит в `TimeScreen_Installer\` — скопируйте эту папку на целевой компьютер.

### На целевом компьютере (Python не нужен):

1. Запустите `install.bat` из папки `TimeScreen_Installer`
2. Программа установится в `%LOCALAPPDATA%\TimeScreen\`
3. На рабочем столе появятся ярлыки:
   - **TimeScreen — Настройки** (главное окно)
   - **TimeScreen — Защита** (вкл/выкл, с паролем)
4. Монитор времени добавится в автозагрузку и запустится

### Настройка:

1. Откройте «TimeScreen — Настройки»
2. Задайте пароль администратора (минимум 4 символа)
3. Добавьте расписание в формате `ЧЧ:ММ-ЧЧ:ММ` через запятую  
   Пример: `08:00-22:00, 14:00-16:00`
4. Нажмите **«Запустить защиту»**
5. В правом верхнем углу появится зелёный индикатор с обратным отсчётом

## Как это работает

1. Фоновый монитор запускается через VBS-скрипт из автозагрузки
2. Каждые 5 секунд проверяет текущее время и конфигурацию
3. Параллельно работает таймер-оверлей (отдельный процесс) — показывает статус
4. Если время вне разрешённых интервалов — запускается **полноэкранная блокировка на всех мониторах**
5. После ввода пароля — **grace-период 10 минут** без повторной блокировки
6. Таймер показывает жёлтым «Отсрочка» пока действует grace

## Архитектура (один .exe, несколько режимов)

```
TimeScreenControl.exe                  → главное окно настроек (GUI)
TimeScreenControl.exe --monitor        → фоновый монитор (headless)
TimeScreenControl.exe --timer          → таймер-индикатор поверх окон
TimeScreenControl.exe --lock           → полноэкранная блокировка
TimeScreenControl.exe install-user     → установка для текущего пользователя
TimeScreenControl.exe uninstall-user   → удаление
```

## Безопасность

| Действие | Требуется пароль |
|---|---|
| Изменение расписания | ✅ Да |
| Вкл/выкл защиты (toggle) | ✅ Да |
| Смена пароля | ✅ Да (старый пароль) |
| Остановка монитора | ✅ Да |
| Разблокировка экрана | ✅ Да |
| Удаление программы | ❌ Подтверждение |

Пароль хранится в `pc_config.json` в виде SHA-256 хэша.  
При утере пароля — удалите `pc_config.json` и задайте новый.

## Примеры расписания

| Интервал | Описание |
|---|---|
| `08:00-22:00` | Доступ с 8 утра до 10 вечера |
| `08:00-12:00, 14:00-20:00` | Утренний + вечерний доступ |
| `22:00-08:00` | Ночной режим (через полночь) |

## Решение проблем

### Экран блокировки не появляется
- Проверьте, что расписание задано и защита включена
- Убедитесь, что монитор запущен (PID-файл существует)
- Проверьте `service.log` в папке программы

### Блокируется только один монитор
- Обновитесь до последней версии — используется `EnumDisplayMonitors` для всех экранов

### После разблокировки сразу блокирует снова
- Работает grace-период (10 минут) — проверьте `pc_config.json` → `grace_until`

### Забыли пароль
- Удалите `%LOCALAPPDATA%\TimeScreen\pc_config.json` и задайте пароль заново

### Не работает таймер-индикатор
- Таймер запускается отдельным процессом. Убедитесь, что `TimeScreenControl.exe` не заблокирован антивирусом.
