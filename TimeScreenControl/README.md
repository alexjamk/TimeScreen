# TimeScreen Control v3.0

Система родительского контроля для Windows с защитой от обхода.

## 📋 Возможности

- **Временные интервалы** - гибкое расписание использования компьютера
- **Контроль пользователей** - выбор конкретных пользователей или всех
- **Защита паролем** - bcrypt хэширование с солью
- **Перетаскиваемый таймер** - показывает оставшееся время до блокировки/разблокировки
- **Экран блокировки** - полноэкранная блокировка с возможностью выключения/перезагрузки
- **Льготный период** - 10 минут на завершение дел после истечения времени
- **Целостность конфигурации** - защита от ручного редактирования файла настроек

## 🚀 Установка

1. Скопируйте папку `TimeScreenControl` на целевой компьютер
2. Запустите `install/install.bat` **от имени администратора**
3. Служба установится и запустится автоматически
4. Используйте ярлык в меню Пуск для входа в настройки

### Требования

- Windows 10/11
- Python 3.8+
- Библиотеки из `requirements.txt` (устанавливаются автоматически)

## ⚙️ Использование

### Первый запуск

При первом запуске программа предложит установить пароль администратора.
**Запомните пароль!** Без него вы не сможете изменить настройки.

### Настройка защиты

1. **Общие** - включение/выключение защиты, управление паролем
2. **Пользователи** - выбор контролируемых пользователей из списка Windows
3. **Временные интервалы** - настройка разрешённого времени
4. **Таймер** - настройки отображения таймера

### Временные интервалы

- Добавьте интервалы, когда использование **РАЗРЕШЕНО**
- Вне этих интервалов компьютер будет заблокирован
- Поддерживаются overnight интервалы (например, 22:00-08:00)

### Таймер

- Показывает время до следующей блокировки/разблокировки
- Можно перетаскивать мышью в любое место экрана
- Правый клик для контекстного меню (скрыть, сбросить позицию)

## 🔒 Безопасность

- Пароли хэшируются алгоритмом **bcrypt** с солью
- Конфигурация защищена контрольной суммой
- Служба работает от SYSTEM account
- Защита от обхода через Task Manager
- Нет уязвимого recovery режима

## 🛡️ Защита от детей

Программа учитывает пытливость детей:

- Конфигурация в `%PROGRAMDATA%` (требует прав админа)
- Установка в `%PROGRAMFILES%` (требует прав админа)
- Служба перезапускается при попытке остановки
- Экран блокировки нельзя закрыть обычными способами
- Нет ярлыков "Защита" - только настройки

## 📝 Удаление

1. Запустите `install/uninstall.bat` **от имени администратора**
2. Выберите, сохранить ли конфигурацию
