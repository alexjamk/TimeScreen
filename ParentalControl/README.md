# Родительский контроль для Windows

Программа для блокировки компьютера по заданному расписанию. Работает **без установки дополнительных программ** - требуется только стандартный Python 3.8+.

## Возможности

- ✅ Блокировка рабочего стола вне заданного времени
- ✅ Настройка нескольких временных интервалов
- ✅ Защита паролем администратора
- ✅ Работа в режиме службы Windows
- ✅ Экран блокировки с возможностью разблокировки, выключения или перезагрузки
- ✅ Защита от завершения процесса (требуются права администратора)
- ✅ Только стандартная библиотека Python (никаких pip install)

## Требования

- Windows 10/11
- Python 3.8 или выше (с python.org или Microsoft Store)
- Права администратора для установки службы

## Установка

### Быстрая установка

1. Скопируйте папку `ParentalControl` на компьютер
2. **Щелкните правой кнопкой** на `install.bat`
3. Выберите **"Запуск от имени администратора"**
4. Следуйте инструкциям мастера установки

### Ручная установка

```batch
# 1. Настройка пароля и расписания
python parental_control.py admin

# 2. Установка службы (от имени администратора!)
python parental_control.py install

# 3. Запуск службы
python parental_control.py start
```

## Использование

### Команды управления

| Команда | Описание |
|---------|----------|
| `admin` | Настройка пароля и временных интервалов |
| `install` | Установка службы Windows |
| `uninstall` | Удаление службы |
| `start` | Запуск службы |
| `stop` | Остановка службы |
| `status` | Показать статус службы |
| `help` | Показать справку |

### Примеры

```batch
# Проверить статус службы
python parental_control.py status

# Изменить настройки (пароль, расписание)
python parental_control.py admin

# Остановить службу временно
net stop ParentalControlService

# Запустить службу
net start ParentalControlService
```

## Как это работает

1. **Служба Windows** работает в фоне и проверяет текущее время каждые 5 секунд
2. Если текущее время **вне разрешенных интервалов**:
   - Показывается полноэкранный экран блокировки
   - Блокируется доступ к рабочему столу
   - Доступны кнопки: "Разблокировать", "Выключить ПК", "Перезагрузить"
3. Для разблокировки требуется ввести **пароль администратора**
4. Когда наступает разрешенное время - блокировка автоматически снимается

## Конфигурация

Файл конфигурации хранится в:
```
C:\ProgramData\ParentalControl\config.json
```

Пример конфигурации:
```json
{
  "password_hash": "abc123...",
  "intervals": [
    {"start": "08:00", "end": "22:00"},
    {"start": "14:00", "end": "16:00"}
  ],
  "created": "2024-01-01T10:00:00",
  "version": "1.0"
}
```

## Безопасность

- Пароль хранится в хэшированном виде (SHA-256 с солью)
- Служба работает с правами SYSTEM
- Обычный пользователь не может завершить процесс без прав администратора
- Экран блокировки нельзя закрыть через Диспетчер задач (требуется пароль)

## Важные замечания

⚠️ **Для обхода блокировки ребенок может:**
- Перезагрузить компьютер в безопасном режиме
- Использовать загрузочную флешку
- Удалить файл конфигурации (если есть доступ)

⚠️ **Рекомендации:**
- Установите пароль BIOS/UEFI
- Отключите загрузку с внешних носителей в BIOS
- Ограничьте права пользователя в Windows

## Удаление

1. **Щелкните правой кнопкой** на `uninstall.bat`
2. Выберите **"Запуск от имени администратора"**

Или вручную:
```batch
net stop ParentalControlService
python parental_control.py uninstall
```

## Решение проблем

### Служба не устанавливается
- Запустите от имени администратора
- Проверьте, что у вас есть права локального администратора

### Экран блокировки не появляется
- Проверьте, что tkinter установлен (входит в стандартный Python)
- Запустите `python parental_control.py run-service` вручную для проверки

### Конфликт с другими программами
- Некоторые программы родительского контроля могут конфликтовать
- Отключите другие подобные программы
