# Родительский контроль для Windows

Программа для ограничения времени использования компьютера детьми. Работает как служба Windows, блокирует рабочий стол вне заданных временных интервалов.

## Два варианта использования

### Вариант 1: Python уже установлен на целевом компьютере

Если на компьютере уже есть Python 3.8+:
- Используйте файлы: `parental_control.py`, `install.bat`, `uninstall.bat`
- Запустите `install.bat` от имени администратора

### Вариант 2: Python НЕ установлен на целевом компьютере ⭐ РЕКОМЕНДУЕТСЯ

Если нужно установить программу на компьютер без Python:

**На компьютере разработчика (где есть Python):**
1. Запустите `create_installer.bat` 
2. Будет создан standalone .exe файл и установочный пакет
3. Скопируйте папку `ParentalControl_Installer` на целевой компьютер

**На целевом компьютере (без Python):**
1. Запустите `install.bat` из папки `ParentalControl_Installer` от имени администратора
2. Готово! Программа работает без Python

## Возможности

- ✅ Гибкое расписание (несколько временных интервалов)
- ✅ Полноэкранный экран блокировки
- ✅ Разблокировка по паролю администратора
- ✅ Кнопки выключения/перезагрузки на экране блокировки
- ✅ Защита от завершения процесса (требуются права администратора)
- ✅ Хэширование паролей (SHA-256 + соль)
- ✅ Автоматический запуск как служба Windows
- ✅ Работа без дополнительных библиотек (только стандартная библиотека Python)

## Установка

### Быстрая установка (если Python уже установлен):

1. Скопируйте все файлы в отдельную папку
2. **Запустите `install.bat` от имени администратора** (правая кнопка → "Запуск от имени администратора")
3. Следуйте инструкциям мастера установки

### Ручная установка:

```bash
# 1. Настройка пароля и расписания
python parental_control.py admin

# 2. Установка службы Windows
python parental_control.py install
```

## Использование

### Панель администратора

Запустите для настройки пароля и расписания:
```bash
python parental_control.py admin
```

Или используйте ярлык "Родительский контроль - Админ" на рабочем столе.

**В меню администратора можно:**
- Установить/изменить пароль администратора
- Добавить временные интервалы разрешения доступа
- Просмотреть текущее расписание
- Очистить расписание
- Включить/выключить защиту

### Команды

| Команда | Описание |
|---------|----------|
| `parental_control.py admin` | Панель администратора |
| `parental_control.py install` | Установить службу Windows |
| `parental_control.py uninstall` | Удалить службу |
| `parental_control.py start` | Запустить службу вручную |
| `parental_control.py help` | Справка |

## Как это работает

1. Служба запускается автоматически при загрузке Windows
2. Каждую минуту проверяет текущее время
3. Если текущее время **вне** разрешенных интервалов:
   - Запускается полноэкранный экран блокировки
   - Блокируется доступ к рабочему столу
   - Отображается сообщение о блокировке
4. Для разблокировки нужно ввести пароль администратора
5. На экране блокировки доступны кнопки:
   - **Разблокировать** (требуется пароль)
   - **Выключить ПК**
   - **Перезагрузить**

## Примеры расписания

**Один интервал:**
- 08:00 - 22:00 (доступ разрешён с 8 утра до 10 вечера)

**Несколько интервалов:**
- 08:00 - 12:00 (утренний доступ)
- 14:00 - 20:00 (вечерний доступ)

**Интервал через полночь:**
- 22:00 - 08:00 (ночной режим)

## Файлы конфигурации

После установки файлы хранятся в:
- `%PROGRAMFILES%\ParentalControl\`

Основные файлы:
- `parental_control.py` - главный скрипт
- `config.json` - настройки (пароль, расписание)
- `service.log` - журнал работы службы
- `run_service.bat` - скрипт запуска службы

## Безопасность

- Пароль хранится в хэшированном виде (SHA-256 + случайная соль)
- Для остановки службы требуются права администратора
- Экран блокировки нельзя закрыть обычным способом
- Конфигурация защищена от несанкционированного изменения

## Удаление

Запустите `uninstall.bat` от имени администратора или выполните:
```bash
python parental_control.py uninstall
```

## Системные требования

### Для запуска готовой программы (вариант 2):
- Windows 10/11
- **НЕ требуется Python** (все включено в .exe файл)
- Права администратора для установки/настройки

### Для сборки .exe файла (только на компьютере разработчика):
- Windows 10/11
- Python 3.8 или выше (нужен только для сборки)
- pip (менеджер пакетов Python)

## Решение проблем

### Как создать .exe файл если Python не установлен?

Вам нужен компьютер с установленным Python ТОЛЬКО для сборки:
1. Установите Python на свой компьютер (https://www.python.org/downloads/)
2. Скопируйте файлы проекта на свой компьютер
3. Запустите `create_installer.bat` 
4. Полученную папку `ParentalControl_Installer` скопируйте на целевой компьютер
5. На целевом компьютере запустите `install.bat` от имени администратора

**Python на целевом компьютере НЕ НУЖЕН!**

### Служба не запускается
Проверьте лог файл `service.log` на наличие ошибок.

### Экран блокировки не появляется
Убедитесь, что:
- Расписание настроено правильно
- Защита включена в панели администратора
- Служба запущена (проверьте в services.msc)

### Забыли пароль администратора
Удалите файл `config.json` из папки установки и настройте пароль заново.
